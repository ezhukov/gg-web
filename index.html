about us

Present Day

Are you facing the uncertainty of how to move forward with this new cloudification thing? Feeling anxious because things can go over budget?

The Choice

Choosing a cloud provider and moving your app to the cloud doesn’t have to be a struggle. Regardless of what you’re trying to achieve as an organization we can help you build the right platform.

Made Unbiased

We do not partner with public cloud providers for money. We help customers choose the right one for their case. We do careful cost benefit evaluation.

Not another DevOps company

Our approach is to focus on your tech stack and gather the data like CPU, memory and storage consumption. We then use this data to pick the most suitable services from a cloud provider. For example we use Azure Retail Prices API and AWS Price List API to programmatically find the best option.

Application First

It is hugely important to assess the application resource consumption and utilization before considering any public cloud.

Cloud Infrastructure

Fit for purpose. No bloat, no upsell because of cloud vendor paid partnerships.

Automation

Flawless and regular application and infrastructure updates as often as needed, as easy as it can be.

Cost & Performance Optimization

Observe, measure and reconfigure the whole solution based on production usage data.

Understanding the whole stack

We deep dive into your application logic and requirements to come up with best in class infrastructure with a suitable cloud provider at optimal cost.
